# PlAnt-Weight-Combat-Robot
CAD of a 600 gram combat robot named guppy masa that i built for a small competition we had between friends
This is a drum styled vertical spinner weaponed bot. We friends had limited our electronics so that everyone has the same base to start off from and hence we used 1000 kv A2212 BLDC Motors, 30 Amps SimonK ESC, SG90 continuous rotation Servo motors, 2200mah 3s 30c lipo batteries and a flysky fsi6 transmitter.
We had a week to CAD the robot in Autodesk fusion.
Everyone had to print their robot completely on a 3D printer with only the PLA filament.
